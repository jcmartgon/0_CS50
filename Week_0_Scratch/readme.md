## About The Project

The goal of the game is to break all the bricks while preventing the ball from reaching the edge of the screen. There's 3 levels total, each more difficult than the last!

### Installation

1) Go to [Scratch](scratch.mit.edu)
2) Click on 'Create' on the navbar
3) Click on 'File'
4) Click on 'Load from your computer'
5) Select 'Sping Spong.sb3' and Loadd

<!-- USAGE EXAMPLES -->
## Usage

- Use the left or right arrow keys to move the horizontal disc accordingly.
- Use the up or down arrow keys to move the vertical disc (level 3 only) accordingly.
